# challenge_amigo_secreto

Amigo Secreto

Descripción

Esta aplicación permite a los usuarios ingresar nombres de amigos en una lista y luego realizar un sorteo aleatorio para determinar quién es el "amigo secreto".

Funcionalidades

Agregar nombres: Permite ingresar nombres a una lista visible en la página.

Validación de entrada: No permite agregar nombres vacíos.

Visualización de lista: Muestra los nombres ingresados en pantalla.

Sorteo aleatorio: Selecciona aleatoriamente un nombre de la lista y lo muestra como resultado.

Instrucciones de Uso

Escribe un nombre en el campo de texto y haz clic en el botón "Añadir".

Verás los nombres agregados en una lista debajo del campo de entrada.

Cuando hayas ingresado todos los nombres, presiona el botón "Sortear Amigo".

Se mostrará en pantalla el nombre del amigo secreto seleccionado aleatoriamente.
